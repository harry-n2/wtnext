# World Trade Next (WTN)

次世代外国人労働者管理システム - AI駆動の国際人材採用と労働力管理のためのプラットフォーム

## 概要

World Trade Nextは、グローバル労働力管理のためのAI搭載効率化プラットフォームです。

### 主な機能

- **Kiwami Zero One Marketing** - 採用マーケティングのための自動化ファネルアプリケーション
- **AI Connector** - LINEメッセージングとAI機能を連携する統合レイヤー
- **Lark Enterprise Suite** - 労働力管理とセキュリティインフラ
- **Services** - AIパートナーシップとコンサルティング
- **Vision** - 企業理念と代表者プロフィール

### 価値提案

- コスト削減: 自動化により約30%削減
- AI効率化: インテリジェントシステムにより約15%向上
- Lark統合: 管理業務で約25%の時間短縮

## デプロイ

このサイトはGitHub Pagesで公開されています。

### ローカル開発

```bash
# 任意のHTTPサーバーで実行

# または
npx serve
```

ブラウザで http://localhost:8000 を開いてください。
